# 랄프톤 — 화면 설계

디자인 톤: 투명, 여백, 콘텐츠 중심. 군더더기 없음.

모바일 퍼스트 375px, 데스크톱 max-width 480px 센터 정렬.

---

## `/` — 원페이지 퍼널

### 랜딩

```
┌──────────────────────────────────────┐
│                                      │
│     뭐든 넣으면                       │
│     당신이 보입니다.                  │
│                                      │
│   ┌────────────────────────────┐     │
│   │                            │     │
│   │   사진 / 글 / 링크 / 메모   │     │
│   │        여기에 drop          │     │
│   │                            │     │
│   └────────────────────────────┘     │
│                                      │
│  ┌────────┐ ┌────────┐ ┌────────┐   │
│  │ 읽기   │ │ 읽기   │ │ 읽기   │   │
│  │ 결과   │ │ 결과   │ │ 결과   │   │
│  │ 예시1  │ │ 예시2  │ │ 예시3  │   │
│  └────────┘ └────────┘ └────────┘   │
│       ← 가로 스크롤 →                │
└──────────────────────────────────────┘
```

#### 이미지 자동 리사이즈

```
사진 drop 즉시:
  Canvas API → max 1920px 리사이즈 → JPEG 0.8
  5MB 초과 → quality 0.6 재압축
  그래도 초과 → max 1280px 추가 리사이즈
  HEIC/HEIF → JPEG 변환 (heic2any)
  실패 → 토스트 "이 사진은 올릴 수 없어요"
```

사용자는 리사이즈를 인지하지 못한다. 1장당 < 1초.

#### drop 엣지케이스

| 케이스 | 처리 |
|--------|------|
| GIF 업로드 | 첫 프레임만 추출 |
| 동영상 선택 | 무시 + 토스트 "사진만 올릴 수 있어요" |
| 갤러리 권한 거부 | 토스트 + 설정 딥링크 |
| 오프라인 | 입력은 가능 (로컬). 분석 버튼만 비활성 |
| 빈 텍스트 전송 | drop 버튼 비활성 |
### 로딩

입력 종류별 로딩 카피:

**사진 drop 시**
```
뭐가 찍혀있는지 보는 중
→ 왜 이걸 골랐을지 생각 중
→ 당신의 말로 옮기는 중
```

**글 / 링크 drop 시**
```
어떤 내용인지 읽는 중
→ 뭐가 걸렸을지 생각 중
→ 당신의 말로 옮기는 중
```

drop한 항목 미리보기를 로딩 화면에 함께 표시.

### 읽기 결과 (대화 진입은 선택)

```
┌──────────────────────────────────────┐
│                                      │
│   ┌────────────────────────────┐     │
│   │   (drop한 항목 미리보기)    │     │
│   └────────────────────────────┘     │
│                                      │
│   이렇게 읽어봤어요                   │
│                                      │
│   "손 닿는 곳의 아늑함을              │
│    모으는 사람"                       │
│   (인사이트 단락)                     │
│                                      │
│   양감    채워진 둥근 볼륨             │
│   질감    도자기 유약, 나무결          │
│   투명도  불투명. 빛을 품는 물질들     │
│   촉각    따뜻하다                    │
│   무게    탁자 위에 올려진 존재감      │
│   온도    미지근 ~ 따뜻               │
│   (관찰 사실 목록)                    │
│                                      │
│   ┌────────────────────────────┐     │
│   │ 뭔가 떠오르면 말해줘요       │     │
│   └────────────────────────────┘     │
│                                      │
│   [ 공유 ]  [ 저장 ]                  │
│                                      │
│            그냥 넘어갈래요            │
└──────────────────────────────────────┘
```

#### 읽기 결과 → 채팅 전환

```
사용자가 자유 입력 필드에 타이핑 + [→] 탭
  ↓
즉시 /chat 으로 전환 (fade 300ms)
  ↓
채팅 화면에 이미 표시됨:
  [읽기 결과 요약 — 상단 고정]
  [사용자] 방금 입력한 반응
  [AI 타이핑 중...]
  ↓
AI 응답 도착 → 바로 표시
  ↓
이후 일반 채팅처럼 자유롭게 주고받기
```

- 입력 필드: 텍스트 비어있으면 [→] 비활성(회색), 입력 있으면 활성
- 전환 후 입력 영역 즉시 활성화
- 읽기 결과 화면의 반응 = 채팅의 첫 번째 사용자 메시지로 세션에 저장
- "그냥 넘어갈래요" 클릭 시 → 읽기 결과만 저장 후 `/home` 이동

## `/chat` — 메타대화

```
┌──────────────────────────────────────┐
│  ← 홈                        프로필  │
│                                      │
│   ┌────────────────────────────┐     │
│   │   (읽기 결과 요약 — 상단)   │     │
│   └────────────────────────────┘     │
│                                      │
│   ┌─────────────────────────┐        │
│   │ AI: 이게 지금 떠오른 게   │        │
│   │ 신기하지 않아요?          │        │
│   └─────────────────────────┘        │
│            ┌─────────────────────┐   │
│            │ 나: 그러게요, 사실   │   │
│            └─────────────────────┘   │
│   ┌─────────────────────────────┐    │
│   │ 발견: "풍경을 좋아하는 게   │    │
│   │  아니라 넓은 곳이 필요한    │    │
│   │  상태구나"                  │    │
│   │              [ 저장할까요? ] │    │
│   │  ┄ 인사이트 블록 인라인 ┄   │    │
│   └─────────────────────────────┘    │
│   ┌────────────────────────┐         │
│   │ 메시지 입력           →  │         │
│   └────────────────────────┘         │
└──────────────────────────────────────┘
```

진입 경로: 읽기 결과 화면에서 사용자가 응답을 입력했을 때.
읽기 결과가 대화 상단에 고정 표시. 인사이트 블록이 대화 흐름 안에 인라인으로 나타남.

#### 채팅 레이아웃 — 고정 입력, 스크롤 메시지

**핵심: 새 메시지가 아래에 추가되고, 자동 스크롤.**

```
┌──────────────────────────────────────┐
│  ← 홈                                │  ← 헤더 (고정)
│  ┌─── 스크롤 영역 ──────────────┐    │
│  │  [읽기 결과 요약 — 상단]       │    │
│  │  [사용자 반응]                 │    │
│  │  [AI 메시지]                  │    │
│  │  [인사이트 카드]               │    │  ← 최신 메시지가 항상 보임
│  │  [AI 타이핑 중...]            │    │
│  └───────────────────────────────┘    │
│  ┌────────────────────────────┐      │  ← 입력 영역 (키보드 위 고정)
│  │ 메시지 입력           📷 [→] │      │
│  └────────────────────────────┘      │
└──────────────────────────────────────┘
```

**채팅 중에는 하단 탭바를 숨긴다.** 나가려면 헤더 `← 홈` 사용.

#### 스크롤 규칙

| 상황 | 동작 |
|------|------|
| 새 AI 메시지 도착 (스크롤이 맨 아래) | 자동 스크롤하여 새 메시지 표시 |
| 사용자가 위로 스크롤한 상태에서 새 메시지 | 자동 스크롤 안 함. "↓ 새 메시지" 플로팅 버튼 |
| 사용자 메시지 전송 | 항상 맨 아래로 스크롤 |
| 키보드 올라올 때 | 마지막 메시지가 보이도록 스크롤 조정 |

#### 전송 후 상태 리셋

```typescript
const sendMessage = async (content, type) => {
  try {
    setIsSending(true);
    // API 호출
  } catch (error) {
    // 에러 처리
  } finally {
    setIsSending(false);
    setAttachedPhotos([]);
    inputRef.current?.focus();
  }
};
```

- 전송 후 동일하게 상태 리셋
- AI 응답 대기 중: 입력은 가능하지만 전송은 불가 (`[→]` 비활성)
- AI 응답 도착 후: `[→]` 즉시 활성화

#### 세션 유지

| 시나리오 | 동작 |
|---------|------|
| 채팅 중 헤더 `← 홈` 탭 | `/home`으로 이동. 세션 저장됨 |
| `/home`에서 다시 채팅 시작 | 24시간 이내: "이어서 이야기할까요?" 배너 표시 |
| 브라우저 닫기 → 재방문 | 마지막 세션 24시간 이내면 이어서 가능 |

채팅 중 탭바가 숨겨지므로 "탭 이동 후 세션 사라짐" 문제 자체가 발생하지 않음.

#### 입력 종류별 AI 첫 반응

| 입력 | 첫 반응 | 탐색 방향 |
|------|---------|----------|
| 풍경 사진 | "왜 풍경만 올렸어?" | 색감/구도/기억 → 현재 상태 연결 |
| 글 링크 | "이 글에서 뭐가 걸렸어?" | 역추적 → 에센스 연결 → 발견 |
| 텍스트 메모 | "이 생각이 왜 지금 떠올랐어?" | 맥락 → 패턴 감지 → 발견 |
| 사물 사진 | "이걸 왜 찍었어?" | 감각 축 분석 → 에센스 연결 |

#### 채팅 엣지케이스

| 케이스 | 처리 |
|--------|------|
| 빈 메시지 전송 | `[→]` 비활성이므로 불가능 |
| 2000자 초과 | 제한 + 카운터 표시 |
| AI 응답 파싱 실패 | "다시 생각해볼게요..." → 자동 재시도 1회 |
| 짧은 응답 ("응", "ㅇㅇ") | AI가 구체적 질문: "좀 더 이야기해줄 수 있어요?" |
| 네트워크 끊김 중 전송 | 로컬 큐 저장 + 빨간 `!` 표시. 탭 시 재전송 |

## `/profile` — 프로필

```
┌──────────────────────────────────────┐
│   나의 에센스                         │
│   "손 닿는 곳의 아늑함을              │
│    모으는 사람"                       │
│                                      │
│   ▶ 6축 펼치기                       │
│   (collapsed by default)             │
│   ┌──────────────────────────┐       │
│   │ 양감   채워진 둥근 볼륨    │       │
│   │ 질감   도자기, 나무결      │       │
│   │ 투명도 불투명              │       │
│   │ 촉각   따뜻하다            │       │
│   │ 무게   탁자 위 존재감      │       │
│   │ 온도   미지근~따뜻         │       │
│   └──────────────────────────┘       │
│                                      │
│   ● ● ● ● ●  (컬러 팔레트)          │
│                                      │
│   (에센스 변화가 있을 시)             │
│   이전: "채워진 둥근 볼륨"            │
│   현재: "넓고 열린 볼륨"      ← new  │
│   ─────────────────────               │
│   나의 주제들                         │
│   공간 · 혼자 · 속도 · ...           │
│   ─────────────────────               │
│   축적된 발견 (5)                     │
│   "준비 안 된 채로 꺼내는 연습"        │
│   "만드는 건 빠른데 닫는 게 어렵다"    │
│   ...                                │
│   ─────────────────────               │
│   반복 키워드                         │
│   #완성 #혼자 #속도 #팀 #공간        │
│   ───  ───  ───                      │
│   홈   주제  프로필                    │
└──────────────────────────────────────┘
```

에센스 한줄 + 6축 collapsible + 팔레트 + 에센스 변화 이력(있을 시) + 나의 주제들 목록 + 축적된 발견 목록 + 반복 키워드 태그.
주제 모음은 리스트 형태.
